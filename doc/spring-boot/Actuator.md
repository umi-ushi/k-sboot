### HTTPエンドポイント

それなりに使えそうなもの

| エンドポイント             | 内容                                  |
|---------------------|-------------------------------------|
| GET /beans          | DIコンテナ管理されたBeanの一覧                  |
| GET /caches         | 利用可能なキャッシュ一覧                        |
| GET /configprops    | `ConfigurationPropertiesのついたプロパティ一覧 |
| GET /env            | 環境変数の一覧                             |
| GET /mappings       | MVC Mappingされた情報                    |
| GET /scheduledtasks | スケジュールされたタスク一覧                      |
